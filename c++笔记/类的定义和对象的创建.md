类是创建对象的模板，一个类可以创建多个对象，每个对象都是类类型的一个变量，创建对象的过程也叫类的实例化。 每个对象都是类的一个具体实例，拥有类的成员变量和成员函数。

> 有些教程将类的成员变量称为类的属性（Property），将类的成员函数称为类的方法（Method）。在面向对象的编程语言中，经常把函数（Function）称为方法（Method）。
>
> 

```c++
class Student {
public:
    //成员变量
    char* name;
    int age;
    float score;
    //成员函数
    void say() {
        cout << name << "的年龄是" << age << "，成绩是" << score << endl;
    }
};
```
