# 顺序容器

容器就是一些特定类型对象的集合

**顺序容器概述**

~~~c++
vector //可变大小数组 支持快速随机访问
    //在尾部之外的位置插入或删除元素可能很慢
deque //双端队列 支持快速随机访问 在头尾插入删除速度很快

list  //双向链表 只支持双向顺序访问 在list中任何位置进行插入删除都很快
    
forward_list	//单向链表 只支持单向顺序访问
    
array 		//固定大小数组 支持快速随机访问 不能添加或删除元素
string		//与vector类似 但专门用于保存字符 随机访问快 在尾部加入删除很快
~~~
